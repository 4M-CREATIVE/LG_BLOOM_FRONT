국가별 난임 시술 출생아 통계

실제 난임시술을 통해
태어나는 아이는 얼마나 될까요?
국내뿐 아니라 국제적으로 난임 시술 출생아 비율이 점차 증가하고 있으며, 우리나라는 2018년 4.2 %로 높은 편으로 나타났습니다.2
그 이후로 정부의 지속적인 난임부부 지원 확대로 전체 출생아 중 난임시술 지원으로 태어난 출생아 수의 비율은 2020년 7%, 2021년 8%, 2022년 9%, 2023년 11%로 지속적인증가 추이를 보이고 있습니다.3
