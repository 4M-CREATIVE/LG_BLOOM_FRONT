난임이란?

난임은, 피임을 하지 않고 정기적인 성생활을 하고 있음에도 불구하고 12개월이 지나도 임신이 되지 않는 상태를 의미합니다.1
일반적인 부부의 경우 피임을 하지 않고 부부생활을 했을 때, 1년 이내에 85-90%가 임신에 성공합니다.2

국내 난임 환자 통계3

국내 난임 환자 수는 2018년 대비 2022년에 4.7% (연평균 1.2 %) 증가했습니다.
2022년 보고된 바에 따르면 전체 난임 환자 중 30-34세가 전체의 36.1%, 35-39세는 전체의 35.7%를 차지했으며 40대 이상 환자도 전체의 약 24%를 차지했습니다.
