예방 접종

임신부와 태어날 아기는 여러가지 질환에 취약합니다. 따라서 임신 시도 전 감염성 질환을 예방하기 위하여 백신 접종을 해 두는 것이 바람직합니다. 특히 생백신(MMR, 수두, 일부 인플루엔자 백신)의 경우 임신 시도 최소 한 달 이전에 접종할 수 있도록 합니다. 남성의 경우에도 태어날 아기와 배우자에게 감염을 최소화하기 위하여 통상적으로 성인에게 권고되는 백신을 접종하는 것이 좋습니다.

생백신: 살아있는 바이러스나 세균을 약독화하여 독성을 제거한 백신

사백신(불활성화 백신): 열이나 화학약품으로 병원균을 비활성 시킨 백신

여성 추천 예방 접종 리스트12

  • 성인용 Tdap 백신 (파상풍, 디프테리아, 백일해)
  • MMR 백신 (홍역, 볼거리, 풍진)*
  • 자궁경부암 백신 (HPV 백신)
  • 수두 백신*
  • A형, B형 간염 백신
  • 인플루엔자 백신 **
  • 수막구균 백신
    (당뇨, 심혈관 질환, 면역저하, 천식 등 고위험인 경우)

*국내 생백신 유통; **불활성화 백신/생백신 모두 사용됨으로 접종시 확인 필요
B형 간염 백신: 질병관리청 주산기감염 예방 사업 링크
