자궁강내 유착이란?

자궁강내 유착은 자궁내막이 광범위하게 파괴된 상태로, 소파수술(자궁안을 긁어내는 수술) 후 흔히 발생합니다.1
자극 혹은 감염 등에 의해 자궁 안쪽의 깊은 층이 손상되고, 그 부위에 염증과 흉터조직이 생기면서 자궁강 내의 앞뒤 벽이 서로 붙게 됩니다. 이렇게 붙은 조직은 점점 굳어지면서 유착이 심해질 수 있습니다.2

무증상인 경우도 많지만 과소월경, 무월경과 같은 월경장애가 있을 수 있고, 착상 유지가 힘들어 난임, 반복유산 등의 문제를 일으킬 수 있습니다.3

자궁강내 유착 치료는 유착을 제거하고 이후에 재발하는 것을 막는 것에 초점이 맞추어져 있습니다. 자궁경을 통해 유착을 제거하고, 유착 방지를 위해 자궁내 장치를 삽입하고 자궁내막 증식을 유도하기 위해 호르몬 투여가 이루어집니다.
